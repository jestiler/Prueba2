Documentación de comandos de GIT

Este html contiene los comandos más usados en git. No es recomendado para quienes no tengan mucha idea sobre esto

Comandos config

Puerto va a empezar a quedar mas negro Este comando sirve para asignar el usuario de git y asi mismo podamos visualizar quien y cuando realiza las acciones.
git config --global user.name "nombre"
Aquí va el texto de información
git config --global user.name “tucorreo@gail.com”
git config --list+
git config --global -l
git config --global alias-l "log"
git config --global alias.s "status"
